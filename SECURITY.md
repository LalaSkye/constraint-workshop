# Security Policy

## Supported Versions
This project supports the `main` branch only unless a release explicitly states otherwise.

## Reporting a Vulnerability
Please do **not** open public issues for suspected security vulnerabilities.

Use GitHub's private vulnerability reporting (Security > Advisories) where available,
or email: ricky.mcjones@gmail.com

## What to Include
- Description of the issue
- Minimal repro steps
- Impact assessment (what could happen)
- Suggested fix (optional)

## Response Expectations
We aim to acknowledge within 7 days and provide an update within 30 days.
