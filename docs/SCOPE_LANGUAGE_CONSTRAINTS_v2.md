# SCOPE LANGUAGE CONSTRAINTS (v2 — Lane Aligned)

## 1. Anchor (use one only)

- "At the translation / classification layer…" (default)
- "At the reporting surface…"
- "From the non-executing layer…"

## 2. Core lines (unchanged)

- "If it can still act, it isn't controlled."
- "No decision → no execution."
- "If it can move, the risk is unchanged."

## 3. Avoid

- "Systems fail because…"
- "The real issue is…"
- "All systems…"

## 4. Safe phrasing

- "At the translation / classification layer, governance fails when…"

## 5. Check

- "Am I describing my layer, or the whole system?"

## 6. Rule

- Strong idea + lane anchor = safe
- Strong idea + no anchor = overlap risk

## 7. Ultra-short

- "At the translation layer — no decision → no execution."
